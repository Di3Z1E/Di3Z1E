## 🔥 Featured Project ~ SpindleX

<div align="center">

### **[SpindleX](https://github.com/Di3Z1E/spindlex)**

**Modern pure-Python SSH library** built for speed, security, and zero-dependency automation.

</div>

**Key Advantages:**
- ~2.6× faster connection times than Paramiko
- Native asyncio support (`AsyncSSHClient` + `AsyncSFTPClient`)
- Secure by default — modern algorithms only (Ed25519, ChaCha20-Poly1305)
- High-performance recursive SFTP with internal buffering
- Fully typed, zero external SSH dependencies
